# ICSTAR 2026 - React Website

A React-based website for ICSTAR 2026 International Conference on Smart Technologies & Advanced Research (Women's Day Edition).

## Features

- **Navigation Bar**: Fixed navigation with smooth scrolling
- **Hero Section**: Eye-catching banner with call-to-action buttons
- **About Section**: Conference information with feature highlights
- **Info Section**: Conference highlights cards
- **Registration**: Pricing tiers for different participant categories
- **Submission Guidelines**: Paper submission requirements and formatting
- **Footer**: Contact information and quick links

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite
